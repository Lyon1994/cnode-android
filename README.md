A [CNode](https://cnodejs.org) client for Android.


## Prerequisites

- Gradle 2.1
- Oracle JDK 7 (8 is recommended)
- Android API Platform 21
- Android SDK Build-tools 21.1
- Android Studio 0.9+ (canary build)


## Build

```sh
./gradlew assembleRelease
```
